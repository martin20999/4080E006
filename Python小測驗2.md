### divmod() 函數
```
divmod() 函數把除數和餘數運算結果結合起來，返回一個包含商和餘數的元組(a // b, a % b)。

在 python 2.3 版本之前不允許處理複數。

函數語法: divmod(a, b)
參數說明： a: 數字 b: 數字

下列結果為何?
divmod(3,5)
輸出:(0,3)
divmod(5,3)
輸出:(1,2)
divmod(-5,3)
輸出:(-2,1)
```

### bin()/oct()/hex() 函數
```
hex()函數用於將10進制整數轉換成16進制，以字串形式表示。
oct()函數用於將10進制整數轉換成8進制，以字串形式表示。
bin() 返回一個整數 int 或者長整數 long int 的二進位表示。


函數語法: hex(x)
參數說明：x -- 10進制整數
返回值:返回16進制數，以字串形式表示。

下列結果為何?
hex(7)
輸出:'0x7'
hex(15)
輸出:'0xf'
hex(16)
輸出:'0x10'
hex(112)
輸出:'0x70'
hex(255)
輸出:'0xff'
hex(255.255)
輸出:錯誤
hex(int(255.255))
輸出:'0xff'
type(hex(12))
輸出:str


oct(10)
輸出:'0o12'
oct(112)
輸出:'0o160'
bin(10)
輸出:'0b1010'
bin(112)
輸出'0b1110000'
```

### int()
```
int() 函數用於將一個字串或數位轉換為整型。

函數語法: int(x, base=10)
參數說明：x -- 字串或數位。     base -- 進制數，預設是十進位。
返回值:返回整數型資料。
 

下列結果為何?
int('12')
輸出:12
int('12',2)
輸出:錯誤
int('12',8)
輸出:10
int('12',16)
輸出:18
int('12',7)
輸出:9
```
